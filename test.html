üîç Troubleshooting

  • Extension not appearing: Make sure you loaded the correct folder and the manifest.json is valid
  • Selection mode not working: Check the browser console (F12) for any error messages
  • Text not highlighting: Ensure the extension toggle is enabled in the popup
  • Popup not opening: Try right-clicking the extension icon and selecting "Open popup"
